Summary:
WiX is a free Windows toolset used to generated Windows install programs.
It provides command lines to generate an install program.
It also includes a Visual Studio extension for creating WiX install projects.
The Windows "Corona.SDK.sln" uses this Visual Studio extension to create Corona's install program.

This tool can be found on the following website:
   http://wixtoolset.org

How to install:
1) Install Visual Studio 2013, if not done already.
2) Run the WiX EXE file.
3) Click the "Install" button on the window.
4) Click the "Exit" button once the install has completed.
