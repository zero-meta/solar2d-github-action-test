Author:		Joshua Quick (Corona Labs Inc.)


Summary:
The "Sandcastle Help File Builder" generates help file and website documentation from Microsoft .NET XML comments. This tool is needed to build Corona Cards for WinRT (ie: Windows Phone and Windows Store Apps).

This tool can be found on the following website:
   https://shfb.codeplex.com

How to install:
1) Install Visual Studio 2013, if not done already.
2) Run the "SandcastleInstaller.exe" file.
3) Click "Next" on all of the steps until installation has been completed.
