CoronaCards for Windows Phone 8

Overview

CoronaCards for Windows Phone 8 provides a Silverlight XAML control and library framework that allows you to easily leverage DirectX to render 2D content and play audio using Lua scripts. This allows you to develop your Windows Phone app in .NET, avoiding the complexities of bridging the C++/CX side of DirectX into your .NET app.

CoronaCards is provided as a Visual Studio Extension (*.vsix) file which integrates Corona's library framework and project templates into Visual Studio.

System Requirements

Desktop operating system required for development:

  • Windows 8.1 (64-bit Pro edition is needed for emulator support)
  • Windows 10 (64-bit Pro edition is needed for emulator support)

Visual Studio versions supported:

  • Visual Studio 2013 (Community, Professional, Premium, and Ultimate)
  • Visual Studio 2015 (Community, Professional, and Enterprise)

Windows Phone operating systems supported:

  • Windows Phone 8.0
  • Windows Phone 8.1
  • Windows 10 Mobile
